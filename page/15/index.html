使用pycharm创建虚拟环境

很多个项目可能要同时使用同一个库，并且多个项目可能会使用不同版本的库。为每个python工程单独创建虚拟环境。

  1. File$\to$New Project。
  2. 在弹出的对话框中，选择New environment using。选择Virtualenv。其他默认
  3. File$\to$Settingts，弹出的对话框中单击Project Interperter，Project$\to$Project Interpreter，点击+弹出的对话框中搜索Flask点击Install Package。

变量规则

<variable_name>，作为关键字参数传递给函数。

<converter:variable_name>，指定转换器。

转换器 作用
string 接收不带斜杆的任何文本(默认)
int 接受整数
float 浮点数
path 表示使用斜杠
any 匹配提供的任意项
uuid UUID字符串

传递HTTP请求

url_for()进行网页重定向跳转到指定的URL。

url_for(endpoint,**valuse)

  • endpoint：要传递的函数名。
  • valuse：以key=value接受endpoint的函数参数。

redirect进行重定向

redirect(location, code: int = 302, Response)

使用session和cookie

cookies

make_response(html_txt).set_cookies('username',values)创建response对象然后设置cookies。

session

http无状态性，设置session可以在服务器开辟一块内存保存Session对象存储特定用户会话所需的属性及配置信息。

在flask框架中，session被加密放到了cookie中。
