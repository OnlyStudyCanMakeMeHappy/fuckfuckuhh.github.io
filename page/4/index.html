MATLAB(矩阵实验室)

矩阵和数组

创建和合并数组

  • 冒号(:)-创建数组

    • 格式：start:step:end
    • step：为负表示增量为负

  • 串联，使用[]将数组连接

    • [,]或者vertcat(A1,...,n)：垂直串联需要有相同的列数
    • [;]或者horzcat(A1,...,An)：水平串联需要有相同的行数

  • rand-均匀分布随机数

    • x = rand：返回区间为(0,1)内均匀分布的随机数
    • x = rand(n)：返回nxn的随机矩阵
    • x = rand(sz1,...,szn)：返回一个指定维度数组
    • x = rand(__,typename)：指定数据类型
    • x = rand(__,'like',p)：创建一个与 p 具有相同大小和数据类型的由随机数组成的数组。

    使用r = a + (b-a).*rand(N,1)​生成指定范围内的随机矩阵。

  • B = blkdiag(A1,...,An)：创建分块对角矩阵

  • repelem-重复元素副本

    • u = repelem(v,n)：重复v中元素对应n次，n为标量或数组。
    • B = repelem(A,r1,...,rn)：重复矩阵每个维度对应r次。r必须是标量或者是与A对应维度具有相同长度的向量

  • repmat-重复数组副本

    • B = repmat(A,n)：返回一个数组，该数组在其行维度和列维度包含A的n个副本，大小是size(A)*n
    • B = repmat(A,r1,...,rN)：指定一个标量列表r1,…,rN，用于表述A的副本在每个维度中如何排列
    • B = repmat(A,r)：使用行向量r指定重复方案，

创建网格

  • linspace(x1,x2,n)：n默认为100，返回x1-x2之间n个网格。间距为$(x2-x1)/(n-1)$

  • logspace-创建对数间距网格

    • y=logspace(a,b)：生成$10^a-10^b$之间50个对数网格点
    • y=logspace(a,b,n)：生成n个对数网格点
    • y=logspace(a,pi,n)：生成$10^a$和pi之间n个点。

  • mershgrid-生成二维、三维网格

    • [X,Y] = meshgrid(x,y)：X 是一个矩阵，每一行是 x 的一个副本；Y 也是一个矩阵，每一列是 y 的一个副本。坐标 X 和 Y 表示的网格有 length(y) 个行和 length(x) 个列。
    • [X,Y,Z] = meshgrid(x,y,z)： 返回由向量 x、y 和 z 定义的三维网格坐标。X、Y 和 Z 表示的网格的大小为 length(y)×length(x)×length(z)。

  • [X1,X2,...,Xn] = ndgrid(x1,x2,...,xn)：生成n维网格

确定大小、形状和排序

  • L = length(x)：返回x最大数组维度长度，等价于max(size(x))

  • size(x)：返回x每个维度的长度

  • ndims(x)：返回x的维度

  • n = numel(A)：返回数组A的元素，等同于prod(size(A))

  • isscaler(A),isvector(A),ismatrix(A),isrow(A),iscolumn(A)：确定A是否为标量，向量，矩阵，行列向量。

  • issorted(A,dim,direction)：确定A沿着维度dim，是否按照direction的顺序排列。

    排序方向

    • ascend：升序
    • descend：降序
    • monotonic：按照升序或者降序排列
    • strictascend：严格升序
    • strictdescend：严格降序
    • strictmonotnic：严格降序或升序

重构和重新排列

  • 删除行或列

    A() = []-只能有一个非冒号索引，即一次必须删除一行或一列

  • sort-对元素排序

    • sort(A,dim,direction)，默认升序排列
      • 如果A是向量，对A向量元素排序
      • 如果A是矩阵，对A的每个列向量进行排序
      • 如果A是多维数组，对沿着不等于1的维度计算
      • direction：’ascend’或’descend’
    • sort(__,Name,value)：指定用于排序的其他参数
      • MissingPlacement-缺失值的位置
        • ‘auto’-升序放在最后，降序放在最前面
        • ‘first’-缺失的元素放在最前面
        • ‘last’-缺失的元素放在最后
      • ComparisonMenthod-元素比较方法
        • ‘atuo’-A为实数时real(A)进行排序，A为复数时按照abs(A)进行排序
        • ‘real’-按照real(A)对A进行排序，实数部分相同比较虚部
      • ‘abs’-按照abs(A)对A进行排序，模相同比较angle(A)

  • flip-数组元素翻转

    • B = filp(A,dim)
    • B = fliplr(A)：从左到右翻转
    • B = flipud(A)：从上到下翻转

  • rot90

    • B = rot90(A)：将A逆时针旋转90°
    • B = rot90(A,k)：将A逆时针旋转90°*k

  • B = permute(A,dimorder)：按照向量dimorder的顺序重新排列数组的维度。matlab与numpy的维度顺序不同，例如三维矩阵在matlab中矩阵个数放在最后一维，numpy则放在第一维

  • A = ipermute(B,dimorder):按照向量dimorder指定的顺序重新排列数组B的维度，使得 B = permute(A,dimorder)。permute的逆运算，已知置换向量和置换后的维度求原维度。

  • B = squeeze(A)：删除长度为1的维度

  • circshift-循环平移数组，行向量向右平移，列向量向下平移

    • Y=circshift(A,K)，如果K为标量则将第一个不为1的维度平移K个位置，如果K为整数向量则将对应维度平移K个位置。

    • Y = circshift(A,K,dim)：将A沿着维度，循环平移数组k个位置。k和dim必须为标量

  • shiftdim-将A的维度数组平移

    • B = shiftdim(A,n)：n为正整数左移，为负整数右移。
    • B = shiftdim(A)：返回数组，元素与A相同，但删除了长度为1的维度。
    • [B,m] = shiftdim(A)：还返回长度为1的维度的数量。

  • reshape-重构数组

    • B = reshape(A,sz)：使用向量sz重构A，prod(sz)必须与numel(A)相等。
    • B = reshape(A,sz1,...,szn)：可以用[]省略一个维度的大小，以便自动计算维度大小。

索引

  • end：终止代码块或者指示最大数组索引，可以在不知道数组长度情况下指示最后一个元素。
    • (:)：索引为列向量
    • (1:end)：索引为行向量
  • colon,:：冒号,用于向量创建、数组下标和for循环迭代。
  • sub2ind(sz,I1,I2)：将下标转换为线性索引
  • ind2sub(sz,ind)：将线性索引转换为下标

变量

  • 显示用过的变量名who，显示变量名和相关信息whois

数据类型转换

  • cast-将变量转换为不同的数据类型

    • B = cast(A,newclass)：newclass是与A兼容的数据类型
    • B = cast(A,'like',p)：将A转换为与p有相同的数据类型和复/实性。

  • typecast(X,type)：将X的位模式转换为type指定的数据类型，而不更改基础数据。它不会更改输入数据。typecast 在输出 Y 中返回的字节数始终与输入 X 中的字节数相同。例如，使用 typecast 将 16 位整数 1000 转换为 uint8 会以两个 8 位段（3 和 232）的形式返回完整的 16 位，从而保持原始值 (3*256 + 232 = 1000)。而 cast 函数将此输入值截断为 255。

运算

算术运算符

MATLA针对矩阵和阵列有两种不同类型运算符

  • 矩阵算术运算
  • 阵列算术运算

运算符

  • +，-：加和减，可以从矩阵中减去任意一个标量。

  • *,.*：矩阵乘法和数组乘法，都可以乘以标量。

    • *：矩阵乘法
    • .*：数组乘法，对应元素相乘

  • /,./：矩阵右除和对应元素相除

    • A/B矩阵右除法表示A*inv(B)，A不能为数组。对线性方程组x*A = B求解。
    • ./表示矩阵对应元素相除，可以是标量，一个行向量和一个列向量会扩展为矩阵。

  • \,.\：矩阵左除

    • A\B表示inv(A)*B，对线性方程组Ax=B求解，A和B有相同的行数。
    • .\是对应元素相除，可以是标量，向量相除会扩展为矩阵。

  • ^,.^：矩阵的幂和数组的幂

    • ^：方阵幂运算，也可以是标量。如果为负数首先反转
    • .^：A.^B表示A的每个元素的B次幂，B可以是标量或者与与A行或列纬度相等的行列向量，作用在A的每一列上的矩阵数组。

  • '和.'：矩阵转置和数组转置，其实两者可以通用。

算术函数

  • cumprod(A,dim)：沿着纬度dim，计算累积。纬度1表示列，纬度2表示行。

  • cumsum(A, dim)：沿着纬度dim，计算累和

  • diff(A,n,dim)：沿着dim计算第n个差分函数

  • prod(A,dim)：沿着dim计算A的积

  • sum(A,dim,'double'/'native'/'default')：沿着dim计算和，native表示与输入一样的类型。

  • round(A)：四舍五入

  • ceil(A)：向正无穷方向舍入

  • fix(A)：向离零方向最近的整数舍入

  • floor(A)：向负无穷方向最近的整数舍入

  • idivide(A,B,OPT)：商取整

    • fix：向零舍入
    • round：四舍五入
    • floor：负无穷舍入
    • ceil：正无穷舍入

  • mod(A,B)：返回A-A.*floor(A./B)

  • rem(X,Y)：返回X-X.*fix(A./B)

  • M=max(A)：返回数组每一列的最大元素

逻辑运算函数

  • all(A,dim)：沿着dim确定数组是否全为非零(负数也行)

  • any(A,dim)：沿着dim确定数组是否有非零数字

  • false(m,n,...)：创建一个逻辑0矩阵

  • true(m,n,...)：创建一个逻辑1矩阵

  • logical(A)：逻辑矩阵转换

  • islogicla(A)：判断是否为逻辑矩阵

  • find(A)：查找非零元素索引，可以传入关于矩阵的布尔表达式

    • find(A)：查找A的所以非零元素索引
    • find(A,k,'last'/'first')：查找A的前k个或者后k个非零元素索引
    • [I,J] = find(X,...)：在稀疏矩阵上运用很有效，返回非零条目的行和列索引
    • [I,J,V] = find(X,...) ：返回X中非零条目的列或行向量v，以及行和列索引

关系运算符

  • eq、ge、gt、le、lt、ne
  • isequal：测试矩阵相等性
  • isequaln：将NAN值视为相等

位运算

  • bitand(a,b)：按位与

  • bitcmp(a,b)：取补码(complement)

  • bitget(a,pos)：获取pos位置的位,pos支持索引

  • bitor(a,b)：按位或

  • bitset(a,pos)：pos位置上设置位

  • bitshift(a,k)：k为正数左移，k为负数右移

  • bitxor(a,b)：逐位移或

集合运算

  • intersect(A,B)：求A、B的交集
  • isnumber(A,B,'rows')：判断B是否是A的成员，默认为列。
  • setdiff(A,B)：差集
  • setxor(A,B)：A和B的异或集
  • union(A,B)：A和B的并集
  • unique(A)：数组中的唯一值
  • join(Tleft,Tright,name,vlue)：按照键值合并表

循环及条件语句

  • break和continue语句后不用加分号;
  • smart index：智能缩进ctrl+I。
  • 使用tic和toc可以实现计时功能。

函数

定义函数

  • 通过edit('which('funcionname.me')')查看函数源代码

  • 定义函数function [y1,...,yN] = myFunc(x1,...,xM)，使用end表示函数结尾

  • 函数返回多个值

函数参数

  • 在一个脚本文件中定义多个函数，其中都是为一个函数所使用。这些函数叫做局部函数仅可供同一文件中的其他函数使用。

  • 函数默认参数变量

    • inputname：函数输入参数的名称

  • nargin：函数输入参数数量

    • nargout：函数输出参数数量
    • varargin：可变长度输入列表，允许函数接受任意数量的输入参数。使用小写字符指定 varargin，将其作为最后一个输入参数附加在任何显式声明的输入项后。在执行函数时，varargin 是一个 1×N==元胞数组==，其中 N 是函数在显式声明的输入后收到的输入项数。但是，如果该函数在显式声明的输入后未收到任何输入，则 varargin 是空元胞数组。

函数句柄

  • 类似c语言的指针，将函数封装成一个变量。

  • 创建函数句柄的格式f = @myfunction

  • 创建匿名函数的句柄，语法为h = @(arglist) anonymous_function

极坐标中绘制线条

  • polarplot(theat,rho)：theta 表示弧度角，rho 表示每个点的半径值。

箱线图和误差图

箱线图

  • boxplot(x)：在每个箱子上，中心标记表示中位数，箱子的底边和顶边分别表示第 25 个和 75 个百分位数。
  • boxchar(ydata)

误差线图

  • errorbar(x,y,err)：绘制 y 对 x 的图，并在每个数据点处绘制一个垂直误差条。

曲面图和网格图

  • peaks：生成包含两个变量的示例函数，peaks 是从高斯分布转换和缩放得来的包含两个变量的函数。

    • Z = peaks：生成一个49x49的矩阵。
    • Z = peaks(n)：返回一个nxn的矩阵。
    • Z = peaks(X,Y)：在给定的 X 和 Y（必须大小相同）处计算 peaks 并返回大小相同的矩阵。

  • surf(X,Y,Z)：创建一个三维曲面图。

  • mesh(X,Y,Z)：创建一个网格图，该网格图为三维曲面。

  • 使用meshgrid创建网格给定X和Y的范围 。
  • surfc(X,Y,Z)：创建一个三维曲面图，其下方有等高线图
  • meshc(X,Y,Z)：创建一个三维网格曲面图，其下方有等高线图

等高线图

  • contour(X,Y,Z)： 创建指定 Z 中各值的 x 和 y 坐标等值线的等高线图，其中 Z 包含 x-y 平面上的高度值。
    • showText：on来显示标签。
  • contourf(X,Y,Z)：填充的二维等高线图。

颜色图

  • colormap(map)：将当前图窗设置为map指定的颜色图。

    • map可以是预定义的颜色，例如：hot、cool等等。

    • 也可以是一个256x3的矩阵，每一行表示一种颜色。

  • colorbar：通过该命令显示色阶的颜色栏。

  • imagesc(C)：将数组 C 中的数据显示为一个图像，该图像使用颜色图中的全部颜色。

多边形

patch- 绘制一个或多个填充多边形区域

  • patch(X,Y,C)：X和Y如果为矩阵，则每一个列向量为一个多边形，C指定多边形的颜色。

  • patch(X,Y,Z,C)：在三维坐标中创建多边形，使用view(3)命令查看。

  • 将Y的最后一项设为NaN，将创建线条，使用Y值为每个顶点定义一种颜色。

颜色

  • flat：每条边使用不同颜色，边颜色取决于顶点的顺序。
  • interp：插入边的颜色，通过在两个边界顶点上进行线性插值来确定边颜色。

注释和标注

  • linespec：线条设定

  • legend()：添加图例

    • legend(label1,...,labelN)/legend(labels)

  • title()和xlabel()/ylabel()：添加标题和标签，支持Latex公式。

  • text(x,y,text)和annotation()

    • 若只有一个文本，则x,y都是标量，否则都是向量，同时字符串要使用字符串的元胞数组或字符串向量数组。
    • annotation()：创建注释，可以指定线条类型lineType用于创建文本箭头注释；形状类型shapeType用于创建文本框注释。
      • shapeType：{‘rectangle’,’ellipse’,’textbox’}，传入四元列表[x_begin y_begin length height]，指定左下角位置和宽度高度。
      • lineType: {‘line’,’arrow’,’doublearrow’,’textarrow’}，传入两个列表[x,y]，[z,w]，指定起点和终点的位置。
      • String：指定文本说明
      • FitBoxText：如果为on，强制文本框紧贴文本

  • axis-设置坐标轴范围和纵横比

      • equal：沿每个坐标轴使用相同的数据单位长度
      • square： 坐标轴线的长度相同。相应调整数据单位之间的增量
      • image：沿每个坐标区使用相同的数据单位长度，并使坐标区框紧密围绕数据。
      • normal：还原默认行为
    • axis ydirection：调整坐标系方向
      • axis ij：反转坐标系，这样 y 的值按从上到下的顺序逐渐增加

数值微积分

多项式

  • polyval-多项式计算
    • y = polyval(p,x)：计算多项式在每个点的值。$p(x) = p1x^n+\dots+p_nx+p{n+1}$

  • polyder-多项式微分

    • k = polyder(p)：返回p中系数表示的多项式导数。$k(x)=\frac{d}{dx}p(x)$

    • k=polyder(a,b)：返回多项式 a 和 b 的乘积的导数，

      $k(x) = \frac{d}{dx}a(x)b(x)$。

    • [q,d]=polyder(a,b)：返回多项式 a 和 b 的商的导数，$\frac{q(x)}{d(x)} = \frac{d}{dx}[\frac{a(x)}{b(x)}]$。

  • conv-二维卷积和多项式乘法

    • conv(u,v)： 返回向量 u 和 v 的卷积。如果 u 和 v 是多项式系数的向量，对其卷积与将这两个多项式相乘等效

  • polyint-多项式积分
    • q = polyint(p,k)：k是积分常量。

数值微分

  • diff-差分和近似导数
    • Y = diff(x)：计算x相邻元素的差分。
    • Y = diff(x,n)：递归应用diff(x)来计算第n个差分，diff(x,2)与diff(diff(x))相同
  • 计算近似导数
