函数

  • 变量搜索顺序

    局部变量->嵌套变量->全局变量->内置变量

    global：声明或定义全局变量

    nonlocal：声明使用嵌套作用域的变量

  • 函数参数

    • 位置参数（必选参数）：

    • 默认参数：指定不变的对象，对于可修改容器默认值指定为None。

    • 可变参数：*args

    • 关键字参数：**kw

    • 命名关键字参数：以分割，\后都是命名关键字参数，必须以参数名=参数值形式传递。每个命名关键字参数都可以指定默认值。

      如果函数中已经有可变参数，则其后的命名关键字参数不再需要分隔符*

      定义默认参数的函数

实现可选参数的函数方法：在函数定义中给参数指定一个默认值，并放到参数列表最后。

  • 减少参数个数

使用partial()函数给参数绑定固定的值，减少调用时参数的个数。

partial()函数常用来当做其他函数的回调函数参数。

多进程和多线程

廖雪峰教程https://www.liaoxuefeng.com/wiki/1016959663602400/1017627212385376

multiprocessing模块

  • 创建进程：调用multiprocessing.Process类构造方法，传入target和args参数分别为目标函数和函数所带的参数以元组形式传入。
  • 进程启动：p.start()自动会调用run方法启动一个进程。
  • join：类似C语言中的wait，主进程等待子进程结束实现进程间的同步。

threading模块

  • Thread类，同样需要target和args
  • start开始线程，join阻塞等待
  • 通过继承Thread类来创建线程，重载run方法。

进程池Pool

  • 通过Pool创建进程池
  • 通过apply_async实现非阻塞异步，即不等待子进程执行完毕，主进程会继续执行。
  • 将所有进程添加到进程池后，调用close方法关闭进程池不允许添加新进程。之后再调用join方法等待所有进程执行完毕。

异步IO

​ 当代码需要执行一个耗时的IO操作时，它只发出IO指令，并不等待IO结果，然后就去执行其他代码了。一段时间后，当IO返回结果时，再通知CPU进行处理。所以一波IO需要一个消息循环，在消息循环中不断重复“等待消息-处理消息”。

asyncio

  • event_loop：开启一个无限循环，将函数注册到时间循环上，当满足某条件时调用相应协程函数。
  • coroutine：协程对象，用async定义的的函数，他的调用不会立即执行函数而是返回一个协程对象。
  • task：对协程对象的封装，包含了任务的各种状态。
  • async/await：关键字，async定义一个协程，await挂起阻塞的异步调用接口。

网络编程

socket连接

  • AF_INET-IPv4，SOCK_STREAM-TCP，SOCK_DGRAM-UDP
  • 通过socket模块创建Socket对象，`def init(self,
             family: int = ...,
         type: int = ...,
         proto: int = ...) -> None`，默认IPv，TCP连接
  • family指定何种IP协议，type选择传输层协议。
  • 连接：connect，通过send(str)发送消息，通过recv(int)接受最大字节的消息，close关闭连接。
  • 服务器端通过bind绑定端口，listen监听，accept以(socket,addr)元组返回一个客服端的连接。
