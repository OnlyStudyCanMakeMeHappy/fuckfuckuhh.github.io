    Bagging和随机森林

    1. Bagging

    使用自助采样法bootstrap sampling，进行m次采样操作。

    自助法

    给定包含m个样本的的数据集D，每次随机从D中挑选一个样本拷贝放入$D^{\prime}$。然后将该样本放入D，使得该样本下次仍可能被取得。整个过程重复m次，就得到一个包含m个样本的$D^{\prime}$。样本在m次采样中始终不被采到的概率为

    自助法在数据集较小时、难以划分训练/测试集时很有用。

    2.Random Forest

    以决策树为基学习器构建Bagging基础上，训练过程引入了随机属性选择。

    在基决策树的每个结点，先从属性集随机选取一个包含k个属性的子集，然后选择一个最优属性进行划分。k的推荐值$\log_2d$。

    基学习器的多样性来自样本扰动(基于Bagging)，属性扰动。

    • 加权平均法

    3. stacking

    先从初级训练集训练出初级学习器，然后生成一个新的数据集通过交叉验证或留一法去训练次级学习器。

    分解

    1.误差-分歧分解

    个体学习器的加权分歧值。

    泛化误差$E$，$\bar{E}$个体泛化器误差的加权平均，$\bar{A}$表示个体学习器的加权分歧值。

    2. 误差-偏差-方差-噪声分解

    • 方差：$var(x)=\mathbb{E}[(f(x)-\bar{f}(x))^2]$，数据的扰动
    • 偏差：$bias^2(x)=(\bar{f}(x)-y)^2$，学习算法的拟合能力
    • 噪声：$\epsilon^2=\mathbb{E}_D[(y_D-y)^2]$

    假设$\mathbb{E}_D[y_D-y]=0$

    提升树

    1. 提升树算法

    算法模型

    算法流程

    • 初始提升树$f_0(x)=0$

    • 第m步的模型

    • 通过经验风险最小化获得第m个决策树的参数

    • 损失函数

      • 回归问题：平方误差损失函数
      • 分类问题：指数损失函数$e^{-y\hat{y}}$

    2. 梯度提升树GBDT

    原理

    回归

    • $f0(x) = \frac{\sum{i=1}^Ny_i}{N}$

    • Loss Functions

      • Absolute loss：$L(y,F)=|y-F|$

      • Huber Loss

    • 梯度提升,由泰勒公式$f(x+\Delta x) = f(x)+{f(x)}’\Delta x $

    • 用回归树去拟合样本$(xi,r{im})$，i=1,2,…,N，分成J个终端区域$R_1,…,R_j$。

    $\nu$学习率0.1

    • 对于每个终端区域的样本，求$\gamma$使该区域的样本损失函数最小化。与回归树该区域的叶子结点均值作为预测值不同，GBRT寻找是该区域样本总损失函数最小的值作为预测值。

    分类

    损失函数

    • 指数损失函数：$exp^{-y\hat{y}}$
    • logLoss(negative binomial log-likehod）：$\log(1+e^{-2y\hat{y}})$，标签为${-1,+1}$
    • Logistic loss：$-{y\log p+(1-y)\log(1-p)}$，标签为0,1

    logistic loss

    sklearn中的集成学习

    Bagging

    sklearn.ensemble.BaggingClassifier(base_estimator=None, n_estimators=10, *, max_samples=1.0, max_features=1 lse, n_jobs=None, random_state=None, verbose=0)-

    参数

    • base_estimator：基估计器，默认决策树
    • n_estimator：base estimates的数量，default=10
    • oob_score：是否使用袋外样本进行泛化误差评价

    属性

    • estimators_：拟合基估计器集合
    • estimatorssamples：每个基估计器使用的样本子集

    随机森林

    sklearn.ensemble.RandomForestClassifier(n_estimators=100, *, criterion='gini', max_depth=None, min_samples_split=2, min_samples_leaf=1, min_weight_fraction_leaf=0.0, max_features='auto', max_leaf_nodes=None, min_impurity_decrease=0.0, min_impurity_split=None, bootstrap=True, oob_score=False, n_jobs=None, random_state=None, verbose=0, warm_start=False, class_weight=None, ccp_alpha=0.0, max_samples=None)

    参数

    • n_estimatorsint：default=100,树的数量
    • max_features：{“auto”, “sqrt”, “log2”}, int or float, default=”auto”，分割时考虑的特征随机子集
    • criterion：{“gini”, “entropy”}, default=”gini”
    • n_jobs：并行运行的作业数，k个作业运行在k个核
    • bootstrap：是否采用自助法建树

    参数调整使用交叉验证以调整到最优

    参数

    • n_estimators：弱学习器的数量
    • learning_rate：控制弱学习器对权重修改速率
    • algorithm：{‘SAMME’, ‘SAMME.R’}, default=’SAMME.R
    • base_estimator：基估计器

    Gradient Tree Booting

    分类

    klearn.ensemble.GradientBoostingClassifier(*, loss='deviance', learning_rate=0.1, n_estimators=100, subsample=1.0, criterion='friedman_mse', min_samples_split=2, min_samples_leaf=1, min_weight_fraction_leaf=0.0, max_depth=3, min_impurity_decrease=0.0, min_impurity_split=None, init=None, random_state=None, max_features=None, verbose=0, max_leaf_nodes=None, warm_start=False, presort='deprecated', validation_fraction=0.1, n_iter_no_change=None, tol=0.0001, ccp_alpha=0.0)

    参数

    • loss：{‘deviance’, ‘exponential’}, default=’deviance’,logistic的对数似然损失
      • deviance：$log(1+e^{-y_if(x_i)})$
      • exponential：$e^{-y_if(x_i)}$
    • n_estimatorsint：default=100，GBT对过拟合有很好的的鲁棒性，所以这个数量越大越好。
    • subsamplefloat：default=1.0，用于拟合各个基学习器的样本比例。＜1导致方差减小，方差增大。
      $err=variance+bias^2+noise$
    • criterion：{‘friedman_mse’, ‘mse’, ‘mae’}, default=’friedman_mse’
    • validation_fraction：预留的训练数据作为验证集

    回归

    sklearn.ensemble.GradientBoostingRegressor(*, loss='ls', learning_rate=0.1, n_estimators=100, subsample=1.0, criterion='friedman_mse', min_samples_split=2, min_samples_leaf=1, min_weight_fraction_leaf=0.0, max_depth=3, min_impurity_decrease=0.0, min_impurity_split=None, init=None, random_state=None, max_features=None, alpha=0.9, verbose=0, max_leaf_nodes=None, warm_start=False, presort='deprecated', validation_fraction=0.1, n_iter_no_change=None, tol=0.0001, ccp_alpha=0.0)

    参数

    • loss：{‘ls’, ‘lad’, ‘huber’, ‘quantile’}, default=’ls’

    • learning_rate：float, default=0.1
