计算机组成原理和结构

1. 冯·洛依曼体系

主要特点

  • 计算机由存储器、运算器、控制器、输入输出设备组成。运算器和控制器称为CPU。

  • 存储器是按地址访问的线性编址的一维结构，每个单元的位数固定。

  • 采用存储程序方式，指令和数据不加区别混合存储在同一个存储器。

  • 控制器通过执行指令发出控制信号控制计算机的操作。指令在存储器中按顺序存放，由指令计数器指明要执行的指令的单元地址。

  • 输入输出设备与存储器之间的数据存放都要经过运算器。

2. 计算机组成部件

存储器

  1. 磁性存储介质。硬盘、磁带等，特点是存储密度高、成本低、具有非易失性，但访存速度慢。

  2. 闪存。非易失性存储介质，与磁盘比访问速度快，成本高，容量小。

  3. DRAM。易失性存储器，存储密度高，周期性刷新，访问速度快。SDRAM构建主存

  4. SRAM。易失性存储器，存储密度不如SDRAM，不同周期性刷新，访问速度快。构建cache

高速缓冲Cache：现代多核处理器通常有多级Cache，一级指令Cache和数据Cache各为几十KB，二级Cache为几百KB，三级Cache为多核共享几MB到几十MB不等。

输入输出设备

  1. GPU图像处理单元。与CPU有大量数据传输，通过DMA方式读取数据，通过解析和运算写至显存。
  2. 硬盘。存放操作系统内核代码、文件系统、应用程序和用户文件数据等。
  3. 闪存Flash。半导体存储器，SD卡、U盘和SSD固态硬盘等。

DDR(Double Data Rate)SDRAM：双倍速率同步静态随机存储器。

3. 计算机硬件结构发展

  • 中央处理器
  • 图形处理器
  • 北桥：离CPU最近的芯片，负责控制显卡、内存与CPU之间数据交换。向上连接CPU，向下连接南桥。
  • 南桥：负责硬盘、键盘以及各种带宽要求较低的IO接口于内存、CPU之间的数据交换。

CPU-GPU-北桥-南桥

CPU、北桥和南桥芯片直接以芯片形式安装在主板，GPU则以显卡形式安装在计算机主板插槽上，通过系统总线和北桥直接相连。

内存控制器集成在北桥芯片，硬盘接口、USB接口、网络接口、音频接口、鼠标和键盘等接口放在南桥芯片。

CPU-北桥-南桥

与四片结构最大的不同是GPU功能被集成到北桥，也就是集成显卡。内存控制器、显示功能集成在北桥芯片。在北桥集成的IO接口速率也不断提高。

CPU-弱北桥-南桥

内存控制器集成到CPU芯片，缓解存储墙问题。

北桥功能被弱化，主要集成GPU、显示接口、高速IO接口（PCIE接口）。

CPU-南桥

CPU芯片集成处理器核、内存控制器、GPU。对外提供显示接口、内存接口等。

南桥芯片则包含硬盘、USB、网络控制器以及PCIE/PCI、LPC等总线接口。

还有一些两片结构把GPU集成在南桥芯片。

SoC单片结构

System on Chip

单个芯片集成处理器、内存控制器、GPU以及硬盘、USB、网络等IO接口。

主要应用在移动处理器和工业控制领域。

计算机系统启动过程

处理器的启动过程，实际上就是一个特定程序的执行过程。这个程序叫做固件，也称为BIOS(Basis Input Output System)。

UEFI(Unified Extensible Firmware Interface)统一可扩展固件接口。

一、BIOS

BIOS被存放在ROM中，系统通电后，处理器第一条指令的地址会被定位到BIOS存储器。在X86系统，这个地址是0xFFFF0。

1.POST

POST(Power-On Self-Test)，硬件自检。硬件出现问题，主板就会出现不同含义的蜂鸣，

2.Bost Sequence

硬件自检完成后，BIOS把控制权交给下一阶段的启动程序。
