调试模式

启用调试支持app.debug=True，则服务器将在代码更改时重新加载自身，如果出现问题，它还将提供有用的调试器。

route装饰器，用于URL注册，绑定url。

run（*host = None*，*port = None*，*debug = None*，*** options* ）

变量规则

<variable_name>，作为关键字参数传递给函数。

<converter:variable_name>，指定转换器。

转换器 作用
string 接收不带斜杆的任何文本(默认)
int 接受整数
float 浮点数
path 接受任意字符串包括包含斜杆
any 匹配提供的任意项
uuid UUID字符串

构造URL

使用url_for()函数构造url，接受函数名称作为第一个参数和许多个关键字参数。
