C++基础

变量和基本类型

类型转换

  • 赋给一个无符号类型一个超出它表示范围的值时，结果是初始值对无符号类型表示数值总数取模后的余数。例如，对于unsigned char表示范围为[0,255]。当把-1赋给8bits大小的unsigned char时所得结果是255。
  • 当给一个带符号类型一个超出范围的值时，结果是未定义的。
  • 不要混用带符号类型和无符号类型。带符号类型会自动地装换成无符号数，会改变有符号负数的取值，有符号数首位作为符号位表示范围在$[-2^{n-1},2^{n-1}-1]$,n表示位数。

变量声明和定义

  • 声明一个变量而不定义它，使用extern关键字标记。
  • 变量只能被定义一次，但可以有多次声明。

类别别名

  • 使用typedef定义类别的别名

编写自己的头文件

  • 头文件通常包含那些只能被定义一次的实体，如类、const和constexpr变量。
  • 头文件也经常包含其他头文件。

预处理器

  • 头文件保护符
    • #define将一个名字设定为预处理变量
    • #ifdef当且仅当变量已经定义时为真
    • #ifndef当且仅当变量未定义时为真
    • 检查结果为真，执行后续操作知道#endif指令为止

constexpr

用于声明常量表达式，即在编译过程就得到计算结果的表达式。

字符串、向量和数组

命名空间的using声明

  • 命名空间是对标识符名称的本地化，避免命名冲突。

  • 使用namespace定义命名空间，通过name::code调用命名空间的函数或变量。

  • 通过using namespace::name指令调用命名空间。

string

构造函数

  • 直接初始化
  • 拷贝构造化

string对象的操作

  • 输入输出流

  • getline(is,s)从is中读取一行赋给s，返回is。遇见换行符为止。

  • s.empay()

  • s.size()

  • 不能通过下标索引添加字符，需要用+运算符等

  • string对象转换为字符串数组

c语言中的name.h头文件在c++中变成cname头文件。

处理string对象中的字符(cctype.h)

  • isalnum：是字母或数字
  • isalpha：是字母为真
  • isdigit：数字时为真
  • islower/isupper：是小写/大写时为真

  • tolower/toupper：字母变小写/大写

  • isspace：空格

for语句处理string

  • 通过引用改变原字符串

vector

如下声明

#include<vector>

定义和初始化vector对象

  • vector<T> v1
  • vector<T> v2(v1)
  • vector<T> v1{a,b,c...}

操作

  • push_back()：向尾部添加元素

  • empty()：是否为空

  • size()：返回向量元素个数

迭代器

通过begin和end成员，分别返回第一个元素的迭代器和尾后元素迭代器(off-the-end)，最后一个元素的下一个位置。

运算符

  • *iter*：获取iter指向的对象
  • iter->men：解引用，取该元素men的成员，等价于(*iter).men。
  • ++,--：
  • ==,!=

迭代器类型

  • 读写迭代器
  • 只读迭代器
